# Mandala Designer

A simple web application for creating beautiful mandala art using rotational symmetry and the HTML5 Canvas API.

## Features

- Create mandalas with configurable symmetry (2–32 axes)
- Kaleidoscope mirror mode
- Smooth drawing experience
- Undo functionality
- Export artwork as PNG


## Built With

- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas API

## Project Structure

```
mandala-designer/
├── index.html
├── style.css
└── script.js

```

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/DishaAndre/mandala-designer.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or dependencies are required.
